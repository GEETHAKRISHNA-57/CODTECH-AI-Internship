# CODTECH AI Internship

This repository contains the tasks completed during the CODTECH Artificial Intelligence Internship.

## Tasks

Task 1: Text Summarization Tool
- Implemented using Python and HuggingFace Transformers.

Task 2: Speech-to-Text System
- Converts speech audio into text using SpeechRecognition library.

Task 3: Neural Style Transfer
- Applies artistic styles to images using deep learning.

Task 4: Text Generation Model
- Generates text using GPT-2 model.

## Technologies Used
- Python
- Transformers
- SpeechRecognition
- PyTorch
